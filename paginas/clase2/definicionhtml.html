HTML (HyperText Markup Language) es el lenguaje estándar para la creación de páginas web y aplicaciones web. Su función principal es estructurar el contenido en la web, organizando texto, imágenes, enlaces, videos, y otros elementos multimedia dentro de un documento. HTML utiliza etiquetas (tags) para definir elementos como párrafos, encabezados, listas, tablas, formularios, y mucho más.

de las características clave de HTML es la capacidad de crear enlaces (hipervínculos) entre páginas, lo que permite la navegación entre diferentes documentos en la web, de ahí el nombre "HyperText" (hipertexto). Aunque HTML se enfoca en la estructura del contenido, puede trabajar junto con CSS (Cascading Style Sheets) para controlar el estilo y la apariencia visual de la página, y con JavaScript para agregar interactividad.
