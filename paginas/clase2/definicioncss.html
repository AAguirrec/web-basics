CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación visual de un documento escrito en HTML o XML. CSS se encarga de controlar cómo se muestran los elementos en la página web, incluyendo aspectos como colores, fuentes, márgenes, espaciado, alineación, y disposición en general.

La principal función de CSS es separar la estructura de un documento (definida en HTML) de su presentación visual, lo que facilita el mantenimiento y la flexibilidad del diseño de la página. Gracias a CSS, se puede aplicar un estilo consistente a múltiples páginas web, y cambiar la apariencia de toda una web editando solo un archivo CSS.
